# Weather Dashboard (React + Vite)

Animated, responsive weather dashboard with:

- city search + autocomplete
- geolocation auto-detect
- current weather metrics
- 24-hour forecast cards
- temperature + rain trend chart
- 7-day forecast
- dynamic weather/day-night backgrounds
- favorites in `localStorage`
- weather map embed

## Data source

- Open-Meteo Forecast API (no API key)
- Open-Meteo Geocoding API (no API key)

## Setup

```bash
npm install
npm run dev
```

No `.env` file is required.

Optional:

```env
VITE_DEFAULT_CITY=New York
```

## Build

```bash
npm run build
```
